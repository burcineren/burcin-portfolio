# Burçin Eren - Software Developer Portfolio

Welcome to my personal portfolio repository. This project showcases my experience, skills, and projects as a Software Developer specializing in .NET and modern Frontend technologies like React, Vue.js, and Angular.

I develop modern and scalable web applications in the Frontend field. I create strong, sustainable code structures with TypeScript and design reusable UI components. With the experience I have gained in e-commerce platforms and corporate projects, I focus on enhancing user experience and improving performance.

- **Frontend:** React, Angular, Vue.js, Next.js, Nuxt.js
- **State Management:** Redux, Pinia, NGXS, React Query
- **Styling:** Tailwind CSS, CSS, Bootstrap, MaterialUI
- **Backend:** .NET, Node.js, Express
- **Tools:** TypeScript, GraphQL, Git, Jira, Figma

## 🏁 Getting Started

To run the project locally:

1. Clone the repository: `git clone https://github.com/burcineren/burcin-portfolio.git`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)
